# YIF Registration System

A web-based registration and authentication system developed for YIF Worldwide.

## 🔧 Features
- User registration with validation
- Secure login system
- Password reset functionality
- Admin dashboard with role control
- AJAX-based form handling (no page reload)

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- PHP
- MySQL
- jQuery (AJAX)

## 💡 Key Highlights
- Handles form validation without clearing user inputs
- Prevents duplicate email registration
- Implements password hashing for security
- Structured for scalability and real-world use

## 📁 Project Structure
- `/css` – Styles
- `/js` – Scripts
- `/images` – Assets
- `register.html`, `signin.html`
- `register_process.php`, `signin_process.php`
